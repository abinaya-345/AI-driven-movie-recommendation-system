# AI Driven Movie Recommendation System

This repository contains an AI-driven movie recommendation system, designed to provide personalized movie suggestions based on user preferences and behavior.

## Features
- Personalized movie recommendations
- User-friendly interface
- Integration with various movie databases

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/abinaya-345/AI-driven-movie-recommendation-system.git
   ```
2. Navigate to the project directory:
   ```bash
   cd AI-driven-movie-recommendation-system
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the application using:
```bash
python app.py
```
